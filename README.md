# Clínica Online 


### Pré-requisitos

- .Net 5 
- Docker  
- Docker-Compose 

### Execução do projeto

- Executar na raiz do projeto:
```bash
docker-compose up -d
```

- Navegar por linha de comando até `src/ClinicaOnline.Web` e executar:
```bash
dotnet run
```

- Acessar `https://localhost:5001/index.html` e a Api deverá está rodando corretamente!
