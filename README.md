# Clínica Online 


### Pré-requisitos

- .Net 5 
- Docker  
- Docker-Compose 

### Execução do projeto

- Executar na raiz do projeto:
```bash
docker-compose up -d
```

- Acessar `http://localhost:5005/index.html` e a Api deverá está rodando corretamente!
